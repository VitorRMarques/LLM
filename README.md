<h1>Jorney</h1>
<p>Meu projeto tem a intencao de ampliar o campo de visao do usuario.</p>
<p>O usuario insere uma lista de tarefas que ele realiza no dia.</p>
<p>O tutor volta a mensagem com a lista de tarefas relacionada ao que o usuário lhe enviou.</p>

<p>My project aims to expand the user's field view</p>
<p>The user insert a list of tasks they perform each day</p>
<p>The tutor replies with a list of tasks related to what the user sent them</p>
