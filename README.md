<h1>Jorney</h1>
<p>Meu projeto tem a intencao de ampliar o campo de viao do usuario atraves e uma mensagem simples, que acontece o desenrolar da ideia.</p>
<p>Um exemplo é: adicionar um começo de uma escrita e o programa lança saidas diversas de palavras que contenham alguma rima ou relação com a palavra anterior.</p>
<p>O usuario insere uma palavra e o programa lança derivadas rimas, sinomimos, antõnimos contextos</p>
<p>Exemplo: palavra inserida: [cenário] -> O programa vai devolver palavras que tenham relacao com essa.</p>
